# DNA Outsourcing Homepage - Images

This folder contains image assets for the DNA Outsourcing homepage.

## Folder Structure

```
images/
├── logos/          # Company and client logos
├── icons/          # UI icons and feature icons
├── backgrounds/    # Background images and patterns
├── blog/           # Blog post thumbnails
├── team/           # Team member photos
└── misc/           # Other images
```

## Image Guidelines

- Use WebP format where possible for better compression
- Provide 2x versions for retina displays
- Optimize all images before adding
- Maximum dimensions: 2000px width for backgrounds, 800px for content images
- Use descriptive filenames: `feature-compliance-icon.svg`
